# Nav list component

Nav list organizes section-level navigation links in a vertical stack.

## Quick example

<UxExampleProvider>
  <MiniNavListExample />
</UxExampleProvider>

## Structure

Group related links under a clear section label when a pane includes multiple destinations.

Use concise item labels so users can scan and move quickly.

## Behavior

The active destination should be visually indicated and remain consistent with the current route.

Nav list items should be keyboard reachable in a predictable top-to-bottom order.
