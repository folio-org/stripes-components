# Filter pane search integration

Filter pane search is a pane-header search pattern for narrowing results inside search-and-filter workflows.

This guidance is integration-focused for module teams.

## Quick example

<UxExampleProvider>
  <MiniFilterPaneSearchExample />
</UxExampleProvider>

## Structure

Filter pane search replaces the pane header area with a search input and clear control.

It may include a skip-to-results focus link to improve keyboard navigation.

## Behavior

Typing updates the search term used for filtering or querying records.

Clear resets the active search quickly.

When a results reference is supplied, keyboard users can jump directly to the results region.
