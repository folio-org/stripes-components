# Info popover component

An info popover provides contextual details about a nearby element without moving users away from the current view.

## Quick example

<UxExampleProvider>
  <MiniInfoPopoverExample />
</UxExampleProvider>

## Structure

An info popover uses an information icon as trigger.

On click, it opens a small overlay containing explanatory content and optional supporting action links.

## Behavior

Place the info icon next to the element it explains.

Selecting the icon opens the overlay. Selecting the icon again or clicking outside closes it.

Use info popovers for clarification, not for primary task actions.
