# Filter groups component

Filter groups organize refinements into clear categories so users can narrow results confidently.

## Quick example

<UxExampleProvider>
  <MiniFilterGroupsExample />
</UxExampleProvider>

## Structure

Group filters by user mental model and keep category names short and specific.

Show related options together and avoid mixing unrelated dimensions in one section.

## Behavior

Applying or clearing a filter should update results predictably and keep current filter state visible.

If filter changes trigger data loading, preserve selection state and communicate progress clearly.
