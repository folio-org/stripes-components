# Menu section component

Menu section groups related menu controls under labeled headings.

## Quick example

<UxExampleProvider>
  <MiniMenuSectionExample />
</UxExampleProvider>

## Structure

Use section labels to separate layout options, column controls, and actions.

Keep destructive actions in a separate section at the end of the menu.

## Behavior

Section labels should stay visible while users navigate menu items with keyboard controls.

Within each section, item order should reflect frequency and task priority.
