# Focus link component

Focus link improves keyboard navigation by letting users skip directly to meaningful targets.

## Quick example

<UxExampleProvider>
  <MiniFocusLinkExample />
</UxExampleProvider>

## Structure

Place focus links before long control groups, dense lists, or repeated navigation blocks.

Use destination labels that describe where focus will move.

## Behavior

Focus links should be reachable by keyboard and activate reliably with Enter.

After activation, move focus to a logical heading or next actionable control.
