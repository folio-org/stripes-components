# Accordion component

The accordion list allows users to show and hide details from a list item.

## Quick example

<UxExampleProvider>
  <MiniAccordionExample />
</UxExampleProvider>

## Structure

Each accordion has content and a title with an arrow on the left allowing users to hide or show content.

If needed, an accordion core information section can be added on top of a record pane accordion set.

## Behavior

If users do not need to see all options or information at once, using an accordion list helps avoid long pages and excessive scrolling.

Clicking the accordion arrow hides or shows detailed information and moves surrounding content upward or downward.
