# Text link component

Text link provides inline navigation actions with accessible link behavior.

## Quick example

<UxExampleProvider>
  <MiniTextLinkExample />
</UxExampleProvider>

## Structure

Use text links for contextual navigation within body copy and metadata areas.

Link text should describe destination or result rather than generic labels.

## Behavior

Text links should remain visually identifiable in all states, including focus and hover.

External links should communicate intent and respect safe target/rel usage when opening new tabs.
