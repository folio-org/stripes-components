# Error modal component

Error modal interrupts the current task to communicate blocking issues and recovery actions.

## Quick example

<UxExampleProvider>
  <MiniErrorModalExample />
</UxExampleProvider>

## Structure

Use a direct title and concise body text that explains what failed and what to do next.

Include one clear dismissal or recovery action before introducing optional secondary choices.

## Behavior

Show error modals only for issues that prevent the user from safely continuing in context.

After dismissal, focus should return to the originating control when possible.
