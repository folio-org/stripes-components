# Error boundary component

Error boundary prevents a failing UI section from crashing an entire page.

## Quick example

<UxExampleProvider>
  <MiniErrorBoundaryExample />
</UxExampleProvider>

## Structure

Wrap high-risk or isolated view sections so failures can be contained and communicated.

Error messaging should explain recovery options in plain language.

## Behavior

When a render error occurs inside the boundary, show fallback content and provide a clear recovery action.

Use reset behavior that returns users to a safe state instead of trapping them on a broken screen.
