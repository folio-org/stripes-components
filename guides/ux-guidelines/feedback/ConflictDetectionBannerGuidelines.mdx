# Conflict detection banner component

Conflict detection banner alerts users when a record changed during editing and requires reconciliation.

## Quick example

<UxExampleProvider>
  <MiniConflictDetectionBannerExample />
</UxExampleProvider>

## Structure

Use concise conflict language that identifies why saving failed and what recovery path is available.

Include a direct link to the latest saved version when users need to compare or reload.

## Behavior

Move focus to the banner when a conflict is detected so keyboard and screen-reader users get immediate feedback.

Keep conflict messaging visible until users resolve or exit the editing flow.
