# Empty message component

Empty message communicates that no content is currently available in a result or detail region.

## Quick example

<UxExampleProvider>
  <MiniEmptyMessageExample />
</UxExampleProvider>

## Structure

Keep empty messages short, specific, and directly tied to the current view context.

When appropriate, pair the message with a next action such as clearing filters or creating a record.

## Behavior

Show empty messages only when data loading has completed and no matching content is available.

Do not present an empty message at the same time as an error state for the same region.
