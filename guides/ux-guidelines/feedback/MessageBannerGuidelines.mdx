# Message banner component

The Message Banner is used to communicate information to the user related to an entire record, section, or group of content.

## Quick example

<UxExampleProvider>
  <MiniMessageBannerExample />
</UxExampleProvider>

## Usage

Use Message Banner when status information applies to a full record area instead of a temporary toast-style notification.

Message content should remain concise and contextual to the section where it appears.
