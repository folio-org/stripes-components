# Paneset integration

Paneset is the layout manager for pane-based module screens.

Use Paneset to manage pane sizing, resizing, and nested pane structures.

## Structure

Use Paneset as the parent container for related panes in a view.

Each pane should have a stable id when you rely on persisted widths or layout transitions.

When nesting panesets inside other panesets, use nested and root options intentionally to avoid layout conflicts.

## Behavior

Paneset tracks child pane widths and preserves user resizing across view state changes.

Use initialLayouts for predictable default width sets.

Use onLayout when pane visibility or app state should drive width changes dynamically.

If a Paneset is rendered inside Layer, mark the layered set as root to prevent unintended ancestor sizing interactions.
