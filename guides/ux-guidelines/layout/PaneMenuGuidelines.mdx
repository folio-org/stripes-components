# Pane menu component

The pane menu groups actions related to pane content and display preferences.

## Quick example

<UxExampleProvider>
  <MiniPaneMenuExample />
</UxExampleProvider>

## Structure

A pane menu is a dropdown action list that can contain one or more sections.

Use separate sections for global list actions, display preferences, or actions on selected records.

Dangerous actions should appear at the end and use destructive styling.

## Behavior

Users open the pane menu from the pane header action control.

Only pane-content actions should be included in the menu. Pane control actions belong to icon buttons.

If an action is not currently available, keep the action visible but disabled and explain why when possible.

## Result and record usage

Result panes often combine actions and display preferences in the same menu.

Record panes typically offer a concise set of actions such as duplicate, bookmark, or delete.
