# Pane subheader component

Pane subheader adds controls directly beneath a pane header without mixing them into the main content flow.

## Quick example

<UxExampleProvider>
  <MiniPaneSubheaderExample />
</UxExampleProvider>

## Structure

Use pane subheader for context switches, segmented controls, or light filter controls that relate to pane content.

Keep subheader content concise to preserve vertical space for records and lists.

## Behavior

Subheader controls should update only the associated pane region.

Subheader state should persist while users interact with content inside the same pane.
