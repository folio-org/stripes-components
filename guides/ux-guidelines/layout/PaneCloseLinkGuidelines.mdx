# Pane close link component

Pane close link provides responsive close and back affordances for pane navigation.

## Quick example

<UxExampleProvider>
  <MiniPaneCloseLinkExample />
</UxExampleProvider>

## Usage guidance

Use pane close link in detail panes that should collapse to a previous list or parent pane.

Ensure the target route or destination preserves user context when returning.

## Behavior

On wider screens, close affordance should support pane dismissal patterns used in split layouts.

On narrow screens, back-style behavior should help users return without losing orientation.
