# Avatar component

Avatar represents user identity in shell and account contexts.

<UxExampleProvider>
  <MiniAvatarExample />

## Location

Avatar placement should be consistent in global profile and identity areas.

## Style

Avatars are compact circular tokens. If no image is provided, a neutral placeholder should appear.

Maintain sufficient contrast between avatar border/background and surrounding UI.
