# Keyboard shortcuts modal component

Keyboard shortcuts modal presents available commands in one place so users can navigate and act efficiently.

## Quick example

<UxExampleProvider>
  <MiniKeyboardShortcutsModalExample />
</UxExampleProvider>

## Structure

Group shortcuts by user intent and keep labels action-oriented.

Use platform-aware key naming so displayed commands match user expectations.

## Behavior

Shortcut help should be discoverable from global UI and dismissible with keyboard controls.

When the modal closes, return focus to the element that opened it.
