# Hot keys component

Hot keys provide efficient keyboard-driven actions for experienced users and high-frequency workflows.

## Quick example

<UxExampleProvider>
  <MiniHotKeysExample />
</UxExampleProvider>

## Structure

Define shortcuts around meaningful action names and keep mappings consistent across modules.

Avoid overloading common keys where browser or assistive technology shortcuts are expected.

## Behavior

Shortcuts should activate only within their intended focus scope.

Expose discoverability through help text or shortcut-reference surfaces for onboarding.
