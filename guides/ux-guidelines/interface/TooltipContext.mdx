# Tooltip context

Tooltips provide short helper text for controls that lack visible labels.

Guidance here is based on local accessibility and usage practices.

## Quick example

<UxExampleProvider>
  <MiniTooltipExample />
</UxExampleProvider>

## Behavior

Tooltips appear on hover or focus and should disappear when focus moves away.

Use short text and avoid placing critical actions inside tooltips.

## Accessibility guidance

Associate tooltip text with the trigger using `aria-labelledby` and `aria-describedby` as appropriate.

Ensure equivalent context is available to keyboard and assistive-technology users.
