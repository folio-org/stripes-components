# Highlighter component

Highlighter emphasizes matched terms inside existing text so users can scan search results faster.

## Quick example

<UxExampleProvider>
  <MiniHighlighterExample />
</UxExampleProvider>

## Structure

Highlight only the terms that come from the current query or filter context.

Keep the original text order and wording unchanged so the content remains trustworthy.

## Behavior

Highlighted terms should update immediately as search input changes.

Highlight styling should preserve contrast and remain readable for long passages.
