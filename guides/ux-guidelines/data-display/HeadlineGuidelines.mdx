# Headline component

Headlines provide hierarchy that helps users scan pages and locate information quickly.

<UxExampleProvider>
  <MiniHeadlineExample />
</UxExampleProvider>

## Location

Use heading levels in hierarchy order from parent sections to nested sections.

In pane-oriented screens, pane-level headings should remain visually prominent while nested section headings are lighter and smaller.

## Typography and style

Headline size and weight should reflect information priority.

Use faded headline styling only for secondary context, not for primary page titles.

Maintain consistent spacing around headings to preserve readable structure.
