# Accordion core info context

Accordion core info is a record-summary section shown outside the accordion set to expose key information immediately.

Modules typically compose this pattern with Headline, KeyValue, and metadata/accordion primitives.

</UxExampleProvider>

## Structure

Core info should appear near the top of a record pane and outside expandable accordion sections.

Depending on record type, core info can include only a primary name/title or a compact set of high-value fields.

## Behavior

Edit and create views should preserve the same information hierarchy as view mode where possible.

Core info should remain stable and scannable so users can quickly confirm record identity.
