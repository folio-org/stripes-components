# Record metadata component

Record metadata provides audit context for a record, including who created or updated it and when.

Record metadata in this codebase is implemented with the MetaSection component.

  <MiniRecordMetadataExample />

## Structure

Record metadata should show creation and last-update details.

Metadata should appear in record view and edit contexts as read-only system-generated information.

The metadata section is typically placed near the top of record details.

## Behavior

Record metadata should be collapsed by default where accordion behavior is used.

When collapsed, summary information should still communicate the latest activity.

Metadata values are system-managed and should not be user-editable.
