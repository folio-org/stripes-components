# Dropdown button component

Dropdown button is a trigger that opens a menu of related actions.

## Quick example

<UxExampleProvider>
  <MiniDropdownButtonExample />
</UxExampleProvider>

## Structure

Use a dropdown button when actions are related but too numerous for direct button placement.

The trigger label should communicate the action category, such as Actions or More.

## Behavior

Selecting the trigger opens an attached menu and selecting an item performs or starts the chosen action.

Menu items should be ordered by frequency and risk, with destructive actions placed last.
