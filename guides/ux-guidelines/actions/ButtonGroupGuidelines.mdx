# Button group component

Button group presents related actions or view switches as an adjacent set of buttons.

## Quick example

<UxExampleProvider>
  <MiniButtonGroupExample />
</UxExampleProvider>

## Structure

Use a button group when actions are closely related and should be compared side by side.

Keep labels short so the grouped controls remain compact and scannable.

## Behavior

Grouped buttons should keep consistent size and spacing across states.

Use a selected/pressed state only when the group acts as a single-choice control.
