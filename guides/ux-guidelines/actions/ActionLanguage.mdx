# Action language

## Duplicating a record

To duplicate a record, the action should be labelled Duplicate.

## Creating a record

To start creating a record, the action should be labelled New and should bring users to the create screen.

## Editing

To start editing a record, the action should be labelled Edit and should bring users to the create screen.

## Saving

To save content or changes, the action should be labelled Save or Save and close if users are redirected to another screen.

## Cancelling

To cancel changes or an action, the action should be labelled Cancel.

## Deleting

To delete a record, the action should be labelled Delete.

## Adding content

When adding content, the action should be labelled as follows: `Add item name in singular`.
