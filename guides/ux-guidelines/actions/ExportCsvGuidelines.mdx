# Export CSV component

Export CSV allows users to extract visible record data for reporting, sharing, or offline analysis.

## Quick example

<UxExampleProvider>
  <MiniExportCsvExample />
</UxExampleProvider>

## Structure

Offer export actions where users expect dataset-level controls, such as result views and lists.

Clarify which fields and records are included before export begins.

## Behavior

Export should preserve column order and labels users see in the interface whenever possible.

When export can take time, provide feedback that the file is being generated.
