# Repeatable field component

A repeatable field allows users to enter multiple values for the same key.

## Quick example

<UxExampleProvider>
  <MiniRepeatableFieldExample />
</UxExampleProvider>

## Structure

To add repeatable fields, place a button in default style with the label format `Add {Item name}`.

## Behavior

When users click Add for the first time, one or more fields in a row should appear.

Only the first repeatable field added should have labels visible.

All repeatable fields should have a trash icon on the right so they can be removed.

When users click Add again, previous field layouts should be cloned below the first set.

No extra labels are needed when adding additional repeatable fields.
