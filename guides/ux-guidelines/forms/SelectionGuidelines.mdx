# Selection component

Selection supports choosing one option from a list with inline filtering.

## Quick example

<UxExampleProvider>
  <MiniSelectionExample />
</UxExampleProvider>

## Structure

Use Selection when option sets are long enough that search filtering improves speed and accuracy.

Option labels should be specific and distinct to reduce ambiguity when filtering.

## Behavior

As users type, option filtering should narrow candidates without changing the selected value until confirmation.

Selection should provide clear keyboard interaction for opening, filtering, and selecting options.
