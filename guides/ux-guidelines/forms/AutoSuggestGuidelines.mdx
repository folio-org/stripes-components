# Autosuggest component

Autosuggest provides type-ahead suggestions based on text input.

## Quick example

<UxExampleProvider>
  <MiniAutoSuggestExample />
</UxExampleProvider>

## Structure

Autosuggest combines a text field with a suggestion list overlay.

The suggestion list should help users confirm likely values while they type.

## Behavior

When users type, matching suggestions should appear.

Users can select a suggestion or continue typing free text based on workflow requirements.

A clear control should remove the current entry and reset the input state.
