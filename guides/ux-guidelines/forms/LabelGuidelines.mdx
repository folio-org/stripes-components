# Label component

Label identifies the meaning of a field or value and anchors form comprehension.

## Quick example

<UxExampleProvider>
  <MiniLabelExample />
</UxExampleProvider>

## Structure

Place labels directly adjacent to their fields using a consistent vertical rhythm.

Use concise label text and avoid punctuation unless required by domain language.

## Behavior

Required and read-only states should be indicated consistently for all labels in a form.

Label text should stay visible when validation errors are shown.
