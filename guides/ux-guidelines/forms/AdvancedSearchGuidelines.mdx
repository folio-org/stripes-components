# Advanced search component

Advanced search supports precise query building when simple search is not enough.

## Quick example

<UxExampleProvider>
  <MiniAdvancedSearchExample />
</UxExampleProvider>

## Structure

Offer clear field options and boolean controls so users can compose complex searches without syntax knowledge.

Use labels that match user vocabulary and domain concepts.

## Behavior

Keep basic search available alongside advanced search so users can escalate only when needed.

When advanced search is dismissed, preserve or clearly reset row state according to page intent.
