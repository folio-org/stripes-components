## Text field

When a field is missing, the error message should state: Please fill this in to continue.

When the input format is wrong, the error message should follow this format: `Field label must contain missing input`.

For other errors, the message should be short and give clear instructions on how to fix the issue. The message should start with the field label.

## Card

When a card has an error, it should be displayed in error style and use the following message format: `Error: Error type + detail to fix it`.

Examples of error messages for uploading files:

- Error: file is too big (max 50mb)
- Error: file format incorrect (CSV only)
- Error: connection issue (check your internet)
