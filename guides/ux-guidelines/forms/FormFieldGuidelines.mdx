# Form field utility

Form field standardizes form state mapping so wrapped inputs behave consistently across form engines.

## Quick example

<UxExampleProvider>
  <MiniFormFieldExample />
</UxExampleProvider>

## Structure

Use form field wrappers to keep input components focused on rendering and interaction concerns.

Map only the validation and state signals needed for user guidance.

## Behavior

Validation messages should appear only after a field is touched or submitted.

Keep wrapper logic predictable so fields respond consistently across modules.
